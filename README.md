#selenium-repl

The most basic REPL for selenium-driver

## Play

	npm install
	npm start

	# repl opens
	.load samples/01-simplest

	# and now type whatever you want
	# you have driver, webdriver and ... in the eval scope....
	
	.save samples/my-magic-sample
	driver.quit();
	.exit


That is it.
